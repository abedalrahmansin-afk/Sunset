# Sunset

A small personal travel web app for the Costa Brava trip.

## Deploy on Vercel
1. Create a GitHub repository called `sunset`.
2. Upload these files to the repository.
3. Go to Vercel, sign in with GitHub, click Add New > Project.
4. Import the `sunset` repository.
5. Keep default settings and click Deploy.
6. Open the Vercel URL on iPhone Safari > Share > Add to Home Screen.
